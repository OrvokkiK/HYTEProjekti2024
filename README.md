# MindEase stressinhallinta sovellus
## Käynnistä Backend-palvelin
```
npm run dev
```
tai
```
npm run start
```

## Required packages
### bcryptjs
```
npm install bcrypt
```
### Chart js
```
npm i chart.js
```

### Chart js plugin zoom
```
npm i chartjs-plugin-zoom
```

### cors (cross-origin resquests)
```
npm install cors
```
### dotenv
```
npm install dotenv
```

### express validator
```
npm install express-validator
```

### mysql2
```
npm install --save mysql2
npm audit fix
```
